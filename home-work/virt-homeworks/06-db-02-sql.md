### Решение задачи 1

```
volumes:

  postgres:
    image: postgres:12
```

|Иоганн Себастьян Бах| Гитара |

## Задача 5

Получите полную информацию по выполнению запроса выдачи всех пользователей из задачи 4 
(используя директиву EXPLAIN).

Приведите получившийся результат и объясните что значат полученные значения.
### Решение задачи 5
```
test_db=# EXPLAIN SELECT c.* FROM clients c JOIN orders o ON c.заказ = o.id;
                               QUERY PLAN
------------------------------------------------------------------------
 Hash Join  (cost=37.00..57.24 rows=810 width=72)
   Hash Cond: (c."заказ" = o.id)
   ->  Seq Scan on clients c  (cost=0.00..18.10 rows=810 width=72)
   ->  Hash  (cost=22.00..22.00 rows=1200 width=4)
         ->  Seq Scan on orders o  (cost=0.00..22.00 rows=1200 width=4)
(5 rows)
```
Прочитана построчно таблица orders, был создан кеш по id для таблицы orders, прочитана таблица clients
 и для каждой строки-полю "заказ" произведена проверка на соответствие лкешу orders:
  - если нет - строка будет пропущена
  - если есть, на основе этой строки кеша СУБД сформирует вывод.
